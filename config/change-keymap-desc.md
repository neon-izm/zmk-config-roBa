# roBa キーマップ解説

## はじめに

roBa は左右分割型の自作キーボードで、右手側にトラックボールが内蔵されています。

このドキュメントでは、**Windows の日本語キーボード配列との共用**を前提としたキーマップ設定を解説します。ノートPC の日本語キーボードとそのまま兼用できる配列を維持しつつ、分割キーボードならではのレイヤーやコンボを活かした構成です。

### 設計方針

- **Windows 日本語キーボード配列に準拠**（ノートPC との兼用を想定）
- **プログラミング用途を優先**した記号配置（`JP_*` マクロで日本語配列のシフト差異を吸収）
- トラックボールをマウス操作・スクロールに活用
- ロータリーエンコーダは使用しない

---

## 用語説明

**レイヤー**: ゲームコントローラーの「シフト状態」のようなもの。特定のキーを押している間だけキーの役割が丸ごと切り替わる仕組み。

**タップ / ホールド**: 同じキーでも「素早く押して離す（タップ）」と「長押し（ホールド）」で別々の動作を割り当てられる。たとえば Space は単押しでスペース入力、長押しで NUM レイヤーへの切り替えになっている。

**モーメンタリー**: 押している間だけレイヤーが有効になる方式。起動したキーを離せば自動でベースレイヤーに戻る。

## レイヤー構成一覧

| # | 名前 | 起動方法 |
|---|------|---------|
| 0 | default | ベースレイヤー |
| 1 | FUNCTION | Delete 長押し |
| 2 | NUM | Space 長押し |
| 3 | ARROW | Enter 長押し |
| 4 | MOUSE | トラックボール移動時に自動遷移（automouse） |
| 5 | SCROLL | P 長押し |
| 6 | BT | Enter+Delete 同時押し |

## 各レイヤーの詳細

### Layer 0：default_layer（普段使い）

日常的な文字入力を行うベースレイヤー。QWERTY 配列をベースに、内側カラムに修飾キーを配置しています。

```
 Q    W    E    R    T              Y    U    I    O    P
 A    S    D    F    G   [Win] [-]  H    J    K    L    ;
Z/⇧   X    C    V    B   [ : ] []] N    M    ,    .    /
Ctrl  ⇧   Alt [半全]  [Spc] [Esc]  BS  [↵]            [Del]
```
        
**ポイント：**

- `Z` キー：タップで `Z`、ホールドで左 Shift（ホームポジションを崩さずにシフトできる）
- ホームロウ内側カラム（左）に `Win` キー（タップで Win、長押しで FUNCTION レイヤー）、（右）に `-`（ハイフン）
- 下段内側カラム（左）に `:`（コロン）、（右）に `]`（右ブラケット）

**親指クラスタの動作：**

| キー | タップ | ホールド |
|------|--------|---------|
| 半/全 | 半角/全角切替 | — |
| Space | スペース | NUM レイヤー（数字・記号） |
| Esc | Escape | — |
| BS | Backspace | — |
| Enter | Enter | ARROW レイヤー（カーソル移動・ウィンドウ操作） |
| Del | Delete | FUNCTION レイヤー（記号・Fn・マウス） |

---

### Layer 1：FUNCTION レイヤー（Delete 長押しで起動）

プログラミング記号・ファンクションキー・マウスボタンをまとめた多機能レイヤーです。

```
 !    "    #    $    %              &    '    (    )    |
 -    ^    ¥    @    _  [→0] [→0] MB3  MB1  MB2  [    ]
F1   F2   F3   F4   F5  [F11] [F12] F6   F7   F8   F9   F10
Ctrl  ⇧   Alt  —    —    —    —    —                   —
```

**ポイント：**

- **上段（Row 0）**：Shift なしで記号を入力
  - 左手：`!` `"` `#` `$` `%`
  - 右手：`&` `'` `(` `)` `|`
- **中段（Row 1）**：
  - 左手：`-` `^` `¥` `@` `_`（よく使うプログラミング記号）
  - 内側カラム（左右）：`→0`（`&to 0` — default レイヤーへ戻る）
  - 右手：MB3（中クリック）・MB1（左クリック）・MB2（右クリック）・`[` `]`
- **下段（Row 2）**：F1〜F12（左に F1〜F5・F11、右に F6〜F10・F12）

> **日本語配列について**: `"` `#` `&` `'` `(` `)` `@` `[` `]` `^` `¥` `_` `|` はすべて `JP_*` マクロで補正済みのため、OS を「日本語キーボード」設定のままでも正しく入力されます。

---

### Layer 2：NUM レイヤー（Space 長押しで起動）

数字と括弧類を配置したレイヤーです。

```
 1    2    3    4    5              6    7    8    9    0
Tab   —    [    ]    —  [   ] [   ]  —    (    )    —    |
 ⇧    —    —    —    —  [   ] [   ]  —    :    '    "    ¥
Ctrl  ⇧   Alt  —    —    —    —    —                   —
```

**ポイント：**

- 上段：数字 `1`〜`0`（横一列でノートPCの数字行と同じ感覚で使える）
- 中段左：Tab、`[` `]`（角括弧）
- 中段右：`(` `)` `|`
- 下段右：`:` `'` `"` `¥`（コード内でよく使う記号）

---

### Layer 3：ARROW レイヤー（Enter 長押しで起動）

カーソル移動と Windows ウィンドウ操作のショートカットを配置したレイヤーです。

```
 —  C+⇧+Tab PrtSc  C+Tab  —          —   Home   ↑   End  —
 —   W+←    W+→   CW+←  CW+→  [  ][  ]  —    ←    ↓    →    —
 —    ⇧      —   S+W+←  S+W+→  [  ][  ]  —   Del  PgUp PgDn  —
 —    —      —    —      —      —    —    —                   —
```

凡例：C=Ctrl、W=Win、S=Shift

**ポイント：**

- **右手**：カーソル移動系
  - `←` `↓` `↑` `→`、`Home`、`End`
  - `Delete`、`Page Up`、`Page Down`
- **左手**：ウィンドウ・タブ操作系
  - `Win+←/→`：ウィンドウを左右にスナップ
  - `Ctrl+Win+←/→`：仮想デスクトップの切り替え
  - `Shift+Win+←/→`：ウィンドウを別モニターへ移動
  - `Ctrl+Tab` / `Ctrl+Shift+Tab`：タブ切り替え
  - `PrintScreen`：スクリーンショット

---

### Layer 4：MOUSE レイヤー（トラックボール移動時に自動遷移）
未使用
---

### Layer 5：SCROLL レイヤー（P 長押しで起動）

キー割り当ては持たず（すべて透過）、トラックボールをスクロール動作に切り替えるためだけのレイヤーです。

P キーを**離すだけで自動解除**されます。ページのスクロールやリスト移動に使用します。

---

### Layer 6：BT レイヤー（Bluetooth 設定）

Enter+Delete の同時押し（コンボ）で遷移します。誤操作防止のため、同時押しでのみアクセスできます。

- **左親指最左（Ctrl 位置）**：`&to 0`（default_layer に戻る）
- 上段右：`BT_SEL 0`〜`BT_SEL 4`（Bluetooth プロファイル 0〜4 に切り替え）
- 下段右端：`BT_CLR`（現在のプロファイルの接続情報をリセット）
- 下段最右端：`BT_CLR_ALL`（全プロファイルをリセット）
- 内側カラム右：`bootloader`（ファームウェア書き込みモード）

> `&to` で遷移したレイヤーは他のキーを離しても自動で戻れないため、左 Ctrl キーを押して明示的に抜けてください。

---

## コンボ（複数キーの同時押し）

コンボキーは、特定の複数キーを同時に押すことで発動する動作です。

| 同時押し | 効果 | 備考 |
|---------|------|------|
| `S`+`D` | Tab | ホームロウ隣接キー |
| `D`+`F` | Shift+Tab | 逆タブ移動 |
| `I`+`O` | `-`（ハイフン） | 右手上段内側 |
| `BS`+`Enter` | `_`（アンダースコア） | 親指クラスタ同時押し |
| `Enter`+`Del` | BT レイヤー（layer 6）へ | Bluetooth 設定 |
| `Q`+`W` | Escape | 左手上段 |
| `Ctrl`+`V` | Undo（Ctrl+Z） | 物理キー同時押し |
| `Ctrl`+`V`+`D` | Redo（Ctrl+Shift+Z） | 3 キー同時押し |

## 日本語配列対応について

このキーマップは、OS を「日本語キーボード」設定のまま使用することを前提としています。日本語配列では US 配列と比べて記号の位置がずれているため、ファームウェア内の `JP_*` マクロでその差異を吸収しています。

| 入力したい文字 | マクロ名 |
|--------------|---------|
| `"` | `JP_DQUOTE` |
| `'` | `JP_QUOTE` |
| `@` | `JP_AT` |
| `:` | `JP_COLON` |
| `[` | `JP_LBRACKET` |
| `]` | `JP_RBRACKET` |
| `(` | `JP_LPAREN` |
| `)` | `JP_RPAREN` |
| `{` | `JP_LBRACE` |
| `}` | `JP_RBRACE` |
| `&` | `JP_AMPERSAND` |
| `^` | `JP_CARET` |
| `¥` | `JP_YEN` |
| `\|` | `JP_PIPE` |
| `_` | `JP_UNDERSCORE` |
| `#` | `JP_HASH` |
| `$` | `JP_DOLLAR` |
| `%` | `JP_PERCENT` |

これにより、FUNCTION・NUM レイヤーの記号キーは OS が「日本語キーボード」に設定された状態でも正しく動作します。
