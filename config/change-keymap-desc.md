# roBa キーマップ説明

## 前提

- Windows の日本語キーボード配列をそのまま使う
- プログラミング用途を優先する
- トラックボールはマウス操作・スクロールに使う（ロータリーエンコーダは使わない）

---

## 用語説明

**レイヤー**: ゲームコントローラーの「シフト状態」のようなもの。特定のキーを押している間だけキーの役割が丸ごと切り替わる仕組み。

**タップ / ホールド**: 同じキーでも「素早く押して離す（タップ）」と「長押し（ホールド）」で別々の動作を割り当てられる。たとえば Space は単押しでスペース入力、長押しで NUM レイヤーへの切り替えになっている。

**レイヤーからの戻り方**: NUM・ARROW・FUNCTION はすべて押している間だけ有効な「モーメンタリー」レイヤー。起動したキー（Space / Enter / Delete）を**離すだけで自動的にベースレイヤーに戻る**。明示的な「戻る」操作は不要。唯一の例外は MOUSE レイヤーで、こちらはトラックボールに触れると自動遷移するため、左手のいずれかのキーを押すと戻る。

---

## レイヤー構成

### default_layer（普段使い）

日常的な文字入力と修飾キー操作を行うベースレイヤー。

```
Q  W  E  R  T          Y  U  I  O  P
A  S  D  F  G [Win][-] H  J  K  L  ;
Z  X  C  V  B  [  ]    N  M  ,  .  /
Ctrl Shift Alt [半/全] Space [Esc] BS Enter Del
```

- `Z` はタップで `Z`、ホールドで `Shift`（左 Shift の代わり）
- 右手内側の `[` `]` は Row 2・3 の内側キー
- **ホームロウ内側（G の右隣）** に `Win` キーを配置

**親指クラスタの動作：**

| キー | タップ | ホールド |
|------|--------|---------|
| 半/全 | 半角/全角切替 | SCROLL レイヤー（トラックボールでスクロール） |
| Space | スペース | NUM レイヤー（数字・記号） |
| Esc | Escape | Win+Shift+S（スクリーンショット） |
| Enter | Enter | ARROW レイヤー（カーソル移動） |
| Del | Delete | FUNCTION レイヤー（ファンクションキー） |

---

### FUNCTION レイヤー（Delete 長押しで起動）

ファンクションキー専用。

- 上段: `F1`〜`F10`、右端: `F11` `F12` `F13`
- 左端: `Escape`

---

### NUM レイヤー（Space 長押しで起動）

数字とプログラミング記号をまとめたレイヤー。日本語配列のずれは内部の `JP_*` マクロで自動補正済み。

- 数字: `0`〜`9`
- 算術: `_` `-` `=` `+` `*`
- 括弧: `(` `)` `[` `]` `{` `}`
- その他: `` ` `` `~` `^` `&` `@` `:` `'` `"` `¥` `|`

---

### ARROW レイヤー（Enter 長押しで起動）

コード編集向けのカーソル移動操作。

- 矢印キー / `Home` `End` / `Page Up` `Page Down`
- `Ctrl+Left` `Ctrl+Right`（単語単位移動）
- `Ctrl+Home` `Ctrl+End`（ファイル先頭・末尾）
- `Ctrl+Tab` `Ctrl+Shift+Tab`（タブ切り替え）
- `Shift+Home` `Shift+End`（行選択）
- `Delete` `Backspace`

---

### MOUSE レイヤー（トラックボール操作時に自動遷移）

トラックボールに触れると自動的にこのレイヤーへ切り替わる。

- 右手: `MB1`（左クリック）`MB2`（右クリック）`MB3`（中クリック）
- 右手: `Shift` `Ctrl` `Alt`（クリックと組み合わせ用）
- 左手: どれかのキーを押すと即座に default_layer へ戻る

---

### SCROLL レイヤー（半/全 長押しで起動）

キー割り当ては持たず、トラックボールをスクロール動作に切り替えるためだけのレイヤー。半/全キーを離すと解除される。

---

## コンボ（複数キーの同時押し）

| 操作 | 効果 |
|------|------|
| `S`+`D` 同時押し | Tab |
| `D`+`F` 同時押し | Shift+Tab |
| `I`+`O` 同時押し | `-` |
| `BS`+`Enter` 同時押し | `_` |
| `Enter`+`Del` 同時押し | layer_6（BT 設定）へ移動 |
| `Q`+`W` 同時押し | Escape |
| `Ctrl`+`V` 同時押し | Undo |
| `Ctrl`+`V`+`D` 同時押し | Redo |
